# 🏦 Bank Customer Segmentation Analysis

## 📌 Overview & Context
This project aims to enhance banking competitiveness and minimize customer churn. By analyzing over **1 million transactions**, I implemented an **RFM (Recency, Frequency, Monetary)** model to segment the customer base. This enables the bank to apply personalized marketing strategies for each segment and optimize marketing budget allocation more effectively.

## 🛠️ Tech Stack
*   **Data Processing:** Python (Pandas, NumPy)
*   **Analytics:** SQL (RFM Modeling)
*   **Visualization:** Power BI, Seaborn, Matplotlib
*   **Documentation:** Markdown & Git

## 🧹 Data Cleaning & Processing
The following steps were taken to ensure data integrity and quality:
*   **Date Standardization:** `DOB` and `TransactionDate` columns were converted to a unified datetime format.
*   **Outlier Removal:** Irrelevant birth years (e.g., 1900) and anomalous transaction amounts were filtered out.
*   **Feature Engineering:** Created new features such as **Age Groups** and **Time-Slots** to identify behavioral patterns.
*   **Handling Null Values:** Missing data was addressed using median imputation or logical filling methods.

## 📊 Key Insights (Executive Summary)
Primary findings from the analysis include:
*   **Revenue Driver:** 60% of total revenue is generated by only 20% of the customer base (**VIP segment**).
*   **The Churn Point:** 15% of customers have been inactive for the last 90 days, with an 85% probability of permanent churn.
*   **Engagement Gap:** Customers aged 25-35 are the most frequent transactors, yet their average balance is 40% lower than other segments.

## 🚀 Business Recommendations

| Category | Strategic Recommendation | Expected Outcome |
| :--- | :--- | :--- |
| **Retention** | Launch targeted cashback offers for customers at the 60-day inactivity threshold. | Reduce churn rate by 10%. |
| **Product** | Introduce "Micro-Investment" products for the 25-35 age group. | Increase Assets Under Management (AUM). |
| **Strategy** | Develop a dedicated Relationship Manager program for VIP customers. | Strengthen customer loyalty and LTV. |
